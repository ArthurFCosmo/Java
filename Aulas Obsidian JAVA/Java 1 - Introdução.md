Sobre o JAVA

- Java foi usado no princípio por que era gratis, e outras tecnologias exigiam que cada programador envolvido no desenvolvimento utilizando-as adquirisse uma licença, o que acabava saindo caro.
- O conceito WORA do Java era muito atrativo, e é usado até hoje.
- Java é uma linguagem estável, novas atualizações permitem escalabilidade deixando suporte para o que já havia sido desenvolvido a tempos. Então, uma empresa que queria um sistema que fosse durar PARA SEMPRE, sem precisar ser totalmente refeito, mirava no Java. E com isso, o Java se tornou bastante popular, sendo a base do sistema de muitas empresas.
- Java é uma linguagem ótima para iniciar, pois utiliza todos os conceitos mais básicos, e explora muito todos eles. Outras linguagens tem isso mais simplificado, então algumas pessoas sentem dificuldade para aprender Java, mas pessoas que vem do Java, aprendem outras linguagens com mais facilidade.
- Tipagem estática.
- Programação orientada a objetos.
- Padronização. Você bate o olho e reconhece o Java, devido à sua alta verbosidade, o jeito como tudo tem que ser escrito. Isso facilita mudanças futuras, quando uma nova equipe for integrada ao projeto para fazer melhorias, reconhecerá facilmente os códigos.
- Java é preferido por sistemas bancários por oferecer segurança, performance, altas demandas, lidar bem com concorrência (duas solicitações ocorrendo ao mesmo instante).
- Java lida bem com a nova era da tecnologia, pois foi adaptado para atender às novas necessidades, como rodar na cloud, mensageria, microserviços.
- Java no mercado de trabalho tem alta demanda, bons salários, longevidade profissional, forte presença em grandes empresas.
- IA auxilia na produtividade, é um instrumento que serve para amplificar a inteligência do usuário. Escalar códigos que ele já sabe, repetir códigos que ele já escreveu, sanar dúvidas rápidas, explicar código legado, sugestão de refatorações, análise de erros, criação de testes.
