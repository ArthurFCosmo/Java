## Quais são?

Os operadores lógicos servem para criar testes lógicos. São eles:

| Operador | Significado    | Exemplo       | Resultado |
| -------- | -------------- | ------------- | --------- |
| &&       | And            | true and true | True      |
| \|\|     | Or             | true or false | True      |
| ^        | Ou um ou outro | true ^ true   | false     |
| !        | Not            | ! false       | True      |
